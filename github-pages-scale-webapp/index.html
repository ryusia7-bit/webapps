      시트 기능 사용

      권한 승인을 받은 뒤 켜면 현재 결과를 구글 시트로 전송할 수 있습니다.

      권한 승인 절차

      • 먼저 권한 승인 페이지를 열어 Google 로그인과 시트 접근 권한을 승인합니다.
      • 대상 스프레드시트에 공유된 Google 계정만 저장 기능을 사용할 수 있습니다.
      • 권한이 없는 계정은 검사, 결과 확인, 인쇄까지만 사용할 수 있습니다.

      아직 구글 시트 전송을 실행하지 않았습니다.

      연동 동작 방식

      • 검사 결과는 웹앱 안에서 계산 즉시 확인할 수 있습니다.
      • 시트 기능 사용이 켜진 경우에만 현재 결과를 구글 시트 DB로 전송할 수 있습니다.
      • 척도 마스터 전송은 문항 정의와 선택지 구조를 시트에 맞춰 갱신합니다.

      GitHub Pages

      공개 배포 안내

      1. 이 폴더 `github-pages-scale-webapp`를 GitHub Pages 배포 대상으로 사용합니다.
      2. 저장소 루트의 `.github/workflows/deploy-pages.yml`이 `main` 브랜치 푸시 때 자동 배포합니다.
      3. GitHub 저장소 설정의 `Pages`에서 Source를 `GitHub Actions`로 지정하면 공개 주소가 활성화됩니다.
      4. 구글 시트 DB 웹앱을 연결하면 저장 기능을 확장할 수 있습니다.

      배포 준비 파일

      • `github-pages-scale-webapp/index.html`
      • `github-pages-scale-webapp/styles.css`
      • `github-pages-scale-webapp/app.js`
      • `github-pages-scale-webapp/questionnaires-data.js`
      • `.github/workflows/deploy-pages.yml`
